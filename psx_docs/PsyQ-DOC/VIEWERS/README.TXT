ABSTRACT: VIEWERS DIRECTORY			07/31/96


This directory contains Adobe(TM) Acrobat(TM) readers and a Microsoft
PowerPoint viewer.  Please choose the needed application and follow 
the respective installation instruction.

	DOS Acrobat Reader			1.0	Acrodos
	Macintosh Acrobat Reader		3.0	Acromac
	Windows Acrobat Reader			3.0	Acrowin
	(16bit or 32bit)

============================
ACROBAT Reader Installation
============================

To view the PDF files on this CD-ROM, you must first install an Adobe 
Acrobat Reader.

Windows Users:
- To install Acrobat Reader 3.0 on your hard drive, run the SETUP.EXE 
  program in the 16BIT (for Windows3.x systems) or 32BIT 
  (for Windows 95 and NT systems) folder for the desired configuration:

	ACROWIN (Acrobat Reader + Search)
		|___ 16BIT		Windows 3.x
		|___ 32BIT		Windows 95 and NT
		|___ QUICKTIM		QuickTime 2.12 Installers
		
- For example, the Acrobat Reader + Search installer for Windows 95 is:
  ACROWIN/ 32BIT / SETUP.EXE
- Follow the instructions on your screen.
- Installation requires approximately 3 MB of free hard disk space. 

* For detailed information on the installation of the Acrobat Reader please
  refer to the readme contained in the directory of the version you select.



Macintosh Users:
- To install Acrobat Reader 3.0 on your hard disk drive, double click on 
  the folder called Acromac. Within the Acromac folder, double click on 
  the folder called Reader. Within the Reader folder, double-click the 
  Install Acrobat Reader3.0 Icon.

	ACROMAC (Acrobat Reader + Search)
		|___ READER		Reader + Search Installer
		|___ OLE		OLE Plug-in Installer
		|___ Quicktime		QuickTime 2.5 Installer

- For example, the Acrobat Reader + Search installer for Macintosh is:
  ACROMAC/ READER / Install Acrobat Reader 3.0
- Follow the instructions on your screen.
- Installation requires approximately 4.5 MB of free hard disk space.

  QuickTime installation is only required if you are viewing PDF files with
  linked movies or sounds.  Most likely, your Macintosh system already has 
  QuickTime 2.0 or later installed, which is required to play back QuickTime
  movies and sounds added to Portable Document Format (PDF) files.

  Installation of the OLE plug-in is only required if you need to view PDF
  documents embedded as OLE objects in another application, such as Lotus Notes.

DOS Users: 
- The Adobe Acrobat 1.0 Readers have been included on this CD ROM. 
- For installation instructions, system requirements and other information 
  please see the readme file located at: 
  [CD-DRIVE]:\DOCS\VIEWERS\ACRODOS\README.TXT
